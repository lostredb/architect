# Get started

This is **Next app** with using libs: drizzle, Elysia, tanstack, Redis

### For launch project in localhost needed print in console commands:
1. `bun i`
2.  `bunx drizzle-kit generate`
3.  `bunx drizzle-kit migrate`
4.  `bun dev`
